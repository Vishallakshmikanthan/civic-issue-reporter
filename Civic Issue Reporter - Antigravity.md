# Civic Issue Reporter - Antigravity
**Official Documentation & Project Report**

## 1. Project Overview
The **Civic Issue Reporter** is a modern, responsive web application designed as a next-generation portal for the **Government of India**. It serves as a bridge between citizens and municipal authorities, facilitating transparent grievance reporting and efficient issue resolution.

The application features a distinct "Digital India" aesthetic with an official Navy Blue, Saffron, and Green color palette, complete with department emblems and a professional dashboard interface.

## 2. Key Features

### 🏛️ Official Government Theme
*   **Branding**: Styled with the official colors of the Indian Government (Navy Blue Primary, Tricolor Accents).
*   **Visual Identity**: Includes simulated Ashok Pillar emblems and department-specific logos.
*   **Typography**: Professional, clean fonts suitable for an official portal.

### 📊 Interactive Analytics Dashboard
*   **Real-time Visualization**:
    *   **Bar Charts**: Animated visualization of issues by category (Road, Sanitation, Safety, etc.).
    *   **Donut Chart**: Live breakdown of grievance statuses (Submitted vs. Resolved).
    *   **Trend Graphs**: Weekly reporting volume analysis.
*   **Data-Driven**: Automatically updates based on the current dataset.

### 📝 Enhanced Grievance Reporting
*   **Smart Form**:
    *   **Secure Evidence Upload**: Drag-and-drop style zone for uploading photos of issues.
    *   **Precise Location**: Dedicated fields for House No, Street, Landmark, and Pincode.
    *   **AI Simulation**: Simulates analyzing the report for categorization logic.

### 🔐 Secure Access Control
*   **Dual Login System**: Separate workflows for 'Citizens' and 'Authorities'.
*   **Credential-Based Access**: Mock secure login requiring ID and Password.
*   **Captcha Simulation**: Visual captcha element for authentic feel.

### 📍 Advanced Navigation
*   **Structured Sidebar**: Collapsible, glassmorphic sidebar for easy navigation.
*   **Nearby View**: Simulated map interface showing issues in the user's vicinity.
*   **Profile Gamification**: Tracks user contributions and resolved issues.

### Tech Stack

## 4. Operational Guide

### For Citizens
1.  **Log In**: Use the "Citizen Login" tab (Any ID/Password works for demo).
2.  **Dashboard**: View active notices and summary stats.
3.  **Lodge Grievance**: Click "Lodge Grievance" or the "+" button. Fill in the details, upload a photo, and submit.
4.  **Track**: Use the "Track Status" or map view to see your report's progress.

### For Authorities
1.  **Log In**: Use the "Page Official" tab.
2.  **Dashboard**: View high-level analytics and department performance.
3.  **Manage**: Click on any issue in the feed to Open it.
4.  **Resolve**: Update status to "In Progress" or "Resolved" and add official comments.
